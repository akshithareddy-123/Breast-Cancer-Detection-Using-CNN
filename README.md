# Breast Cancer Detection Using CNN

This project is a deep learning based Breast Cancer Detection System developed using CNN (Convolutional Neural Network) and Flask.

## Features
- Detects breast cancer from histopathology images
- Predicts whether the tissue is Benign or Malignant
- Displays cancer stage prediction
- Shows estimated survival rate
- User-friendly Flask web application

## Technologies Used
- Python
- Flask
- TensorFlow / Keras
- CNN
- HTML/CSS
- NumPy

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the Flask application

python app.py

3. Open browser and visit

http://127.0.0.1:5000/
